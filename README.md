- vue2
 - 有的：基础练习及过渡有的
 - 没有的：JSX,服务端渲染，自定义指令是没有的 
- Vue-router
 - 有的： 基础路由，命名路由，视图，别名，重定向
 - 没有的：过渡(和vue2大同小异)，钩子函数的实现，路由元，数据获取，滚动行为什么的【单纯的HTML/JS不好模拟嗯】

# 跑Demo

1. 先全局安装一个静态服务器

`npm install  -g anywhere`

2. 然后安装下项目的依赖

`npm install`

3. 在项目下启动静态服务器[【anywhere说明文档】](https://www.npmjs.com/package/anywhere)

`anywhere 8858`
